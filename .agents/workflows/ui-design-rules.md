UI Design Rules for CampusOS AI:

Design the app in a very clean, minimal, professional style inspired by GPay-like simplicity and modern fintech apps.

Use:
- dark and light themes
- very limited colors
- soft neutral backgrounds
- one main accent color only
- borderless cards
- large rounded corners
- strong spacing
- clean typography
- minimal icons
- subtle shadows or elevated surfaces instead of heavy borders
- simple, premium, distraction-free layouts

Avoid:
- too many colors
- neon UI
- excessive gradients
- hard borders around cards
- cluttered dashboards
- too many icons
- too many chart colors
- visually noisy elements

Design language:
- minimal institutional SaaS
- premium mobile-first
- calm, clear, and trustworthy
- polished like GPay / finance app cleanliness

Student and faculty app should feel clean and lightweight.
Admin web app should feel professional, structured, and minimal.
